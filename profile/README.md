# Viscept: Tales of AI

We build open tools for AI-native creation, worldbuilding, and creative workflows.

Our work is builder-oriented, remix-friendly, and practical. We like tools that help people create faster, iterate better, and turn ideas into reusable systems.

## Open Source

Some of the projects we share publicly:

- [`neta-skills`](https://github.com/talesofai/neta-skills) - tools and skills for AI-assisted creative workflows
- [`ComfyBrowser`](https://github.com/talesofai/comfyui-browser) - tools for browsing and working with ComfyUI-related assets and workflows

We keep things simple: useful tools, clean interfaces, and room for the community to build with us.

## Contributing

We welcome:

- bug reports
- documentation improvements
- examples and tutorials
- integrations
- thoughtful pull requests

If you want to contribute, start with the README and open issues in the repo you care about most. Small, focused contributions are great.

# 看见概念

我们在做面向 AI 原生创作、世界构建和创意工作流的开源工具。

我们的风格偏向 builder：重视可复用、可 remix、可持续迭代，也更关心工具是否真的能帮人更快地创作、更稳定地把想法变成作品和系统。

## 开源项目

这里是我们公开贡献的一些项目：

- [`neta-skills`](https://github.com/talesofai/neta-skills) - 面向 AI 辅助创作工作流的工具与技能集合
- [`ComfyBrowser`](https://github.com/talesofai/comfyui-browser) - 用于浏览和处理 ComfyUI 相关资产与工作流的工具

我们喜欢做简单、好用、可扩展的工具，也欢迎社区一起把它们变得更好。

## 如何参与

欢迎你通过这些方式参与：

- 提交 bug
- 改进文档
- 补充示例和教程
- 提供集成方案
- 提交认真打磨过的 PR

如果你想参与，建议先从对应仓库的 README 和 issues 开始。小而清晰的贡献同样非常有价值。
